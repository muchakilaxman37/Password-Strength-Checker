# Password-Strength-Checker

A simple and secure Password Strength Checker Web App built using Python Flask.
This project helps users check wheather their password is Weak,Medium,or Strong based on different security rules.

# Features

Check password strength instantly
Detect:

Uppercase letters
Lowercase letters
Numbers
Special characters
Password length

Beautiful and simple UI
Real-time password analysis
Beginner-friendly Flask project

# Technologies Used 

Python
Flask
HTML
